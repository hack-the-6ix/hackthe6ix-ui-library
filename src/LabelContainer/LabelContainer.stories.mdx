A container used for labelling form components and/or sections with descriptions

<Props of={LabelContainer} />

## Basic usage

<Preview>
  <ToReact vue={LabelContainer} name='twt' label='owo'>
    <input name='twt' placeholder='text'/>
  </ToReact>
</Preview>

## With description
<Preview>
  <ToReact vue={LabelContainer} name='twt' label='owo' description='uwu'>
    <input name='twt' placeholder='text'/>
  </ToReact>
</Preview>

## With required
<Preview>
  <ToReact vue={LabelContainer} name='twt' label='owo' required>
    <input name='twt' placeholder='text'/>
  </ToReact>
</Preview>
